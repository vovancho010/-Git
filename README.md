МОЯ ИНСТРУКЦИЯ ПО GIT 

Git — это система контроля версий. Она хранит снимки (snapshots) всей папки проекта в каждый момент времени, когда ты делаешь коммит.

Три состояния файла в Git:
- Modified (изменён) — ты поменял файл, но Git ещё не знает об этом (не отслеживает).
- Staged (подготовлен) — ты сказал Git: "этот файл я хочу включить в следующий коммит" (команда git add).
- Committed (зафиксирован) — данные сохранены в локальной базе Git (команда git commit).
